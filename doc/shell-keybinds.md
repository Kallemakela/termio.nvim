# bash and zsh binds for moving and editing text

| Key               | Action                                                          |
| ----------------- | --------------------------------------------------------------- |
| `C-a`             | start of line                                                   |
| `C-e`             | end of line                                                     |
| `C-b`             | back one char                                                   |
| `C-f`             | forward one char                                                |
| `M-b`             | back one word                                                   |
| `M-f`             | forward one word                                                |
| `C-d`             | delete char under cursor                                        |
| `C-h` / Backspace | delete char before cursor                                       |
| `C-w`             | delete word before cursor                                       |
| `M-d`             | delete word after cursor                                        |
| `C-u`             | delete from cursor to start of line                             |
| `C-k`             | delete from cursor to end of line                               |
| `C-y`             | paste last killed text                                          |
| `M-y`             | cycle killed text after `C-y`                                   |
| `C-_` / `C-x C-u` | undo                                                            |
| `C-l`             | clear screen                                                    |
| `C-r`             | reverse history search                                          |
| `C-s`             | forward history search, may be blocked by terminal flow control |
| `C-p`             | previous history                                                |
| `C-n`             | next history                                                    |
| `M-.` / `M-_`     | insert last argument of previous command                        |
| `C-t`             | transpose chars                                                 |
| `M-t`             | transpose words                                                 |
